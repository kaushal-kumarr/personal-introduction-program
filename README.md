# Personal Introduction Program

This is a beginner Python project that asks the user for personal information and displays a friendly welcome message.

## Features
- Takes user input
- Uses variables
- Displays formatted output
- Beginner friendly Python program

## Technologies
Python

## How to Run

1. Install Python
2. Run the file

python personal_intro.py

## Sample Output

What is your name? Alex
How old are you? 21
What is your favorite hobby? Coding

Welcome Alex!
You are 21 years old and love Coding.
