# Personal Introduction Program

This is a beginner Python project that asks the user for personal information and displays a friendly welcome message.

## Features
- Takes user input
- Uses variables
- Displays formatted output
- Beginner friendly Python program

## Technologies
Python

## How to Run

1. Install Python
2. Run the file

python personal_intro.py

## Sample Output

What is your name? Kaushal Kumar
How old are you? 20
What is your favorite hobby? Chess

Welcome Kaushal Kumar!
You are 20 years old and love Chess.
